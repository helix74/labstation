# Speckit Checklist Workflow

1.  **Context**: Determine what needs validation (e.g., "Code Quality", "UX Review", "Security").
2.  **Generate**:
    - Create a markdown checklist based on `constitution.md` and industry best practices.
    - Example items: "All functions have types", "No hardcoded secrets", "Mobile responsive".
3.  **Output**: Save as `checklist.md` or present in chat for immediate use.
